Responsive re-order test

Test description

This page should showcase what happens when the order of a page is changed from mobile to desktop.

On larger screens, the image and description alternate left and right but on smaller screens the image is always first. Screen-readers read from top to bottom though so this could lead to a confusing experience for users.
