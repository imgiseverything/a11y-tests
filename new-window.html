a11y tests

New window link test

Links that open in a new window (or tab, most likely a tab) should inform the user that they are doing that.

Success criteria

  1. There is a warning spoken in assistive technology that this link opens to a new window.
  2. There is a visual warning in text that this link opens to a new window.
