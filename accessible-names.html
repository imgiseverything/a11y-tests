Accessible names test

Buttons

The idea is that we want to display simple buttons that say 'Add' but we want to also provide context to a screen-reader about what 'Add' refers to.

If we use aria-describedby then the screen-reader should read out something like Ticket ticket 1, Add button or Add button, Ticket ticket 1

and if we use aria-label then the screen-reader should read out whatever we value we use.
